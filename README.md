### Requirements

- [HiRID](https://physionet.org/content/hirid/1.1.1/).

### Installing

Clone this repository.

Cd into it.

``` bash
pip3 install -r requirements.txt
```
Update paths inside config.json according to yours.

### Usage

Run the notebooks in the correct order.
